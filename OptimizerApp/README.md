# OptimizerApp

Aplicación de escritorio para Windows 11 orientada al monitoreo y optimización de rendimiento.

## Requisitos previos
- Windows 11
- .NET SDK 8 o 9
- Visual Studio 2022 con el workload ".NET desktop development"
- Windows 10/11 SDK
- Git

## Primeros pasos
1. Instala el .NET SDK y Visual Studio con el workload indicado.
2. Crea la solución y el proyecto WPF (se añadirá en el siguiente paso de setup).
3. Ejecuta la aplicación desde Visual Studio o con `dotnet run`.

## Pruebas
Una vez creado el proyecto de tests, ejecutar: `dotnet test`.
